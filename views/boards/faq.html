  • 배송관련
    업데이트 하자마자 구매했는데 바로배송 가능한가요?
    입금 후 공장측 원단문제로 더 이상 제작이 어려운 경우 저희 98도씨에서는 빠른 배송을 위해 품절 상품 취소처리 도와드리고 있는 점 양해부탁드립니다
  • 기타관련
    적립금 예치금은 어떻게 쓰나요?
    ▶ 적립금
    - 상품금액 30,000원 이상 구매 시 사용 가능
    - 적립금 1회 사용 시 2,000원 부터 사용 가능
    - 적립금 + 쿠폰 동시 사용 불가

    ▶ 예치금
    - 1원부터 사용 가능
    - 적립금+예치금 동시사용 가능
    - 쿠폰 + 예치금 동시 사용 가능
    - 모든 환불 처리는 초기 결제해 주신 수단 또는 예치금 환불 처리만 가능합니다.
    (예치금으로 환불 진행 시 환불 철회 불가 한 점 참고 부탁드립니다 )
